Название программы: Photo Album

Данная программа написана на Python 3.5

Данные о запуске: в коммандной строке откройте директорию, где высохранили фаил и введите ...>python photo_album.py

Данная программа отображает фотографии. Выделяет похожие фотографии.

На вашем компьютере должны быть установлены:
1)Python 3.5 - https://www.python.org
2)PyQt5 - https://www.riverbankcomputing.com/software/pyqt/download5
3)PIL
